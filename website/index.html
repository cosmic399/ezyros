Why EzyROS

Everything handled.
Nothing left to break.

From a fresh Ubuntu install to a fully working ROS2 environment — fully automated.

🔍

Smart Detection

Auto-detects your Ubuntu version and shows only the ROS2 distros compatible with your system. No wrong installs, ever.

🔁

Safe to Re-run

Run it 10 times — it cleans up previous installs, stale configs, and broken Python packages before starting fresh every time.

🛡️

Pre-flight Checks

Verifies root access, internet connection, disk space, and apt lock before touching anything on your system.

📋

Full Logging

Every action logged to ~/.ezyros/install.log — know exactly what happened if something goes wrong.

🗑️

Clean Uninstall

Re-run the script and type uninstall — removes every package, repo, key, and bashrc line. Your workspace stays safe.

⚡

Idempotent bashrc

Shell environment lines are added only once using exact-match guards. Run the script 100 times — your bashrc stays clean.

Security

Built to be trusted.

Every safety check runs before the script touches your system.

🚫

No Root Execution

Refuses to run as root. Uses sudo only for specific commands that need it.

💾

Disk Space Check

Verifies at least 5GB free before starting. Fails fast with a clear message.

🔒

apt Lock Guard

Waits up to 60s for apt to free up. Never corrupts package state.

✅

Install Verified

Checks setup.bash exists after install. If not found — fails with a clear error.

📝

Open Source

Every line of the script is public on GitHub. Read it before you run it.

🌐

Official Sources Only

Packages from packages.ros.org only. No third-party PPAs or sketchy mirrors.
